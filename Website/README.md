# Charlottesville Bach

Marketing site for Charlottesville Bach, a bachelorette-party decorating service.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Deploying

This is a standard Next.js App Router project — deploy by importing the repo at [vercel.com/new](https://vercel.com/new), or run `npx vercel` from this directory.
